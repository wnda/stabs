stabs.js

What?

stabs.js is about 50 lines of JavaScript for creating tabbed sections of your webpage.

Why?

Bootstrap requires jQuery. Other alternatives tend to require jQuery. Vanilla JS alternatives navigate to hash fragments in the URL bar.

I didn't want those things.

How?

stabs.js uses only vanilla JavaScript primarily relying on while loops and classList.

IE?

IE9, or IE8 if you polyfill classList.

I experimented with className but the code was horribly ugly and inefficient.
